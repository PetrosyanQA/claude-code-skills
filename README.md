# Claude Code Skills — MPSTATS

Скиллы для AI-агентов поверх MPSTATS API. После установки агент сам работает с аналитикой маркетплейсов и генерирует визуал под карточки — без написания кода.

## Установка

```bash
npx skills add https://github.com/mpstats-io/claude-code-skills -y -g
```

## Скиллы

### mpstats

Полный справочник MPSTATS API для аналитики Wildberries, Ozon и Яндекс Маркет.

**122 эндпоинта, 30+ готовых скриптов.**

Что умеет агент:
- Анализ категорий, ниш, брендов, продавцов — продажи, выручка, тренды, ценовая сегментация
- SKU-аналитика: история продаж, остатки по складам и размерам, позиции в поиске, отзывы
- Сравнение периодов, похожие товары, сезонность
- Генерация аналитических отчётов в фирменном стиле MPSTATS

**Требуется токен MPSTATS API.** Получить в личном кабинете: [mpstats.io/userpanel](https://mpstats.io/userpanel) (блок "API token"). Токен передаётся агенту через переменную окружения `MPSTATS_TOKEN`.

### photo-editor

AI-генерация визуала под маркетплейс-карточки селлеров через внутренний MPSTATS Photo Editor API.

**Покрывает основные операции: фотосессии, инфографика, замена/удаление фона, апскейл, перекраска, "товар в действии", произвольный prompt-edit.**

Что умеет агент:
- Сделать фотосессию товара 1..8 кадров в нужном стиле и размерах
- Сгенерировать инфографику или фотосессию для карточки WB/Ozon 
- Поставить товар на стоковый студийный фон или в сцену использования
- Перекрасить товар в фирменный hex-цвет
- Удалить фон, увеличить разрешение
- Произвольный prompt-edit с референсными изображениями

Двухшаговые пайплайны (`test → generate`) обёрнуты в одну команду `auto` — агент за один вызов получает финальный результат.

**Требуется токен MPSTATS API** (тот же, что и для основного скила) — передаётся в заголовке `X-Mpstats-TOKEN`, сохраняется в `photo-editor/config/.env`.

## Как устроены скиллы

Оба скилла построены одинаково:

```
<skill>/
├── SKILL.md          — точка входа, индекс
├── scripts/          — bash-скрипты для API-вызовов
├── references/       — детальные контракты (API-методы / стадии дизайна)
└── config/           — настройки и токен
```

`SKILL.md` загружается в контекст агента как индекс. Скрипты из `scripts/` агент вызывает напрямую (без написания кода). Файлы из `references/` подгружаются по необходимости — контекст не забивается, но доступ к любому методу API и гайду есть.
